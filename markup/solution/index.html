    Адаптивная верстка

    Интернет-магазин одежды Seller создан с учетом мульти-платформенного использования. Это значит, что независимо от того, какое устройство используют ваши посетители: персональный компьютер, планшет или смартфон, ваш магазин будет отображать информацию максимально удобно и сохранит все функциональные возможности.

    Маркетинговые возможности

    Для эффективной работы с посетителями сайта вы сможете использовать самые разные инструменты:

    • Баннеры;
    • Рекомендации;
    • Новинки;
    • Хиты продаж;
    • Акции;
    • Подписки;
    • Комментарии;
    • Оценки.

    Поиск по сайту

    Seller предлагает несколько способов навигации по сайту, настроив которые, вы сможете показывать вашим клиентам максимально точные предложения.

    • Морфологический поиск;
    • Тэги;
    • Каталог по типу продукции;
    • Каталог по брендам;
    • Фильтр по параметрам;
    • Сортировка каталога.
